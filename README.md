# line-sticker-crawler

A **Line Sticker** downloader by python

## Install Environment
```=shell
sh bin/install-env.sh
```

## Start Download
```=shell
sh bin/start-download.sh 'https://store.line.me/stickershop/product/331/zh-Hant'
```

* Python Packages
	* requests
	* BeautifulSoup
	* lxml
	* Pillow
